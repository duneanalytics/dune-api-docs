## balances_bnb.bep20_latest 
Latest token balances of bep20 bnb tokens per wallet and contract address pair. Depends on bep20_bnb_transfers.


<ParamField path="blockchain">
Blockchain
</ParamField>
<ParamField path="last_updated">
UTC timestamp when data was last updated
</ParamField>
<ParamField path="wallet_address">
Address of the wallet holding this bep20 token
</ParamField>
<ParamField path="token_address">
Contract address for the bep20 token
</ParamField>
<ParamField path="amount_raw">
Raw amount of bep20 token held *before* taking into account token decimals
</ParamField>
<ParamField path="amount">
Raw amount of bep20 token held *after* taking into account token decimals
</ParamField>
<ParamField path="amount_usd">
Amount of bep20 token held in USD if we have it
</ParamField>
<ParamField path="symbol">
Token symbol
</ParamField>
