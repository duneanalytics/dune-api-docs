## balancer_v2_polygon.lbps 
Record of the Liquidity Boostrapping Pools (LBP) of Balancer, an automated portfolio manager and trading platform, on Polygon.


<ParamField path="name">
Name of the LBP
</ParamField>
<ParamField path="pool_id">
ID of the LBP
</ParamField>
<ParamField path="token_sold">
Contract address of the token sold by the LBP
</ParamField>
<ParamField path="token_symbol">
Symbol of the token sold by the LBP
</ParamField>
<ParamField path="start_time">
LBP start time
</ParamField>
<ParamField path="end_time">
LBP end time
</ParamField>
