## optimism_attestationstation_optimism.events 
A table containing creation events of AttestationStation on Optimism.


<ParamField path="block_date">
Block date of the creation event.
</ParamField>
<ParamField path="tx_hash">
Transaction hash of the creation event.
</ParamField>
<ParamField path="block_number">
Block number.
</ParamField>
<ParamField path="block_time">
Block timestamp.
</ParamField>
<ParamField path="evt_index">
Event index.
</ParamField>
<ParamField path="recipient">
Who the issuer is attesting about.
</ParamField>
<ParamField path="issuer">
Who is attesting.
</ParamField>
<ParamField path="contract_address">
Contract address.
</ParamField>
<ParamField path="key_raw">
Hashed key of the attestation.
</ParamField>
<ParamField path="key">
What is being attested about.
</ParamField>
<ParamField path="val_raw">
Hashed value of the attestation.
</ParamField>
<ParamField path="val">
What is the value of the attestation.
</ParamField>
<ParamField path="val_byte2numeric">
For Error Handling: What is the value of the attestation - In bytea2numeric representation (if relevant format).
</ParamField>
<ParamField path="val_string">
What is the value of the attestation - Array converted to a comma separated string.
</ParamField>
