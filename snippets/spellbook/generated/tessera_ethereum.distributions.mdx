## tessera_ethereum.distributions 
Rae auction/distribution started in Tessera. It contains all available auction styles (LPDA, etc).


<ParamField path="id">
Auction ID
</ParamField>
<ParamField path="type">
Type of Auction. For example, LPDA (Last Price Dutch Auction)
</ParamField>
<ParamField path="token">
Generated token address
</ParamField>
<ParamField path="info">
Further information about the distribution
</ParamField>
<ParamField path="vault">
Address of the Vault
</ParamField>
<ParamField path="block_time">
Timestamp when the action took place
</ParamField>
<ParamField path="tx_hash">
Transaction hash for the transaction in which the action took place
</ParamField>
