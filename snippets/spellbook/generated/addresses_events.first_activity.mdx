## addresses_events.first_activity 
Table showing first activity of each address on every chain

<ParamField path="blockchain">
Name of the blockchain
</ParamField>
<ParamField path="address">
Concerned address
</ParamField>
<ParamField path="first_activity_to">
First transaction activity to address
</ParamField>
<ParamField path="first_block_time">
The block time on which the address made it&#x27;s first activity
</ParamField>
<ParamField path="first_block_number">
The block number on which the address made it&#x27;s first activity
</ParamField>
<ParamField path="first_tx_hash">
The transaction hash on which the address made it&#x27;s first activity
</ParamField>
<ParamField path="first_function">
first function called by address
</ParamField>
