## zeroex_polygon.api_fills 
0x API erc20 swaps raw fills (including multihops) and also native swaps thru 0x exchange contracts (without using 0x API) on polygon


<ParamField path="block_month">
UTC event block month of each trade
</ParamField>
<ParamField path="evt_index">
Index of the corresponding order filled event
</ParamField>
<ParamField path="tx_hash">
Transaction hash of the fill
</ParamField>
<ParamField path="maker">
seller of the nft
</ParamField>
<ParamField path="taker">
buyer of the nft
</ParamField>
<ParamField path="matcher">
address/bot that brings two public quotes (from buy as well as from the sell side) together
</ParamField>
<ParamField path="nft_address">
NFT project address
</ParamField>
<ParamField path="nft_id">
Token ID of the NFT project address
</ParamField>
<ParamField path="label">
type of nft - ERC 721 OR ERC 1155
</ParamField>
<ParamField path="price_label">
token address used to genearate the amount in usd - for eth, we take the price of weth
</ParamField>
<ParamField path="token">
token used to pay for the transaction
</ParamField>
<ParamField path="token_amount_raw">
raw token amount used to pay for the transaction
</ParamField>
<ParamField path="symbol">
symbol of the token used to pay for the transaction - use this column for all your descriptive needs + aggregations
</ParamField>
<ParamField path="price_usd">
price of a transaction in USD
</ParamField>
<ParamField path="token_amount">
token amount, after dividing by the correct decimal
</ParamField>
<ParamField path="project_name">
name of the nft project
</ParamField>
<ParamField path="tx_from">
Address which initiated the trade
</ParamField>
<ParamField path="tx_to">
Address which received the trade
</ParamField>
<ParamField path="blockchain">
Blockchain which the aggregator project is deployed
</ParamField>
