## labels.op_trades_derived_archetype 
- Labels starting with &quot;Avid&quot;, &quot;Elite&quot;, &quot;Voracious&quot; and &quot;Habitual&quot; are in the top 5% of the specified dataset (e.g. minters, traders, voters e.t.c) based on transaction volume, trade volume, mint volume e.t.c.

- Labels starting with &quot;Active&quot; are in the top 35% (this excludes those in the top 5%) of the specified dataset based on transaction volume, trade volume, mint volume e.t.c.

- Labels starting with &quot;Normie&quot; and &quot;Casual&quot; are in the bottom 65% (i.e. those not in the top 35% or top 5%) of the specified dataset based on transaction volume, trade volume, mint volume e.t.c.

- &quot;OP Governance Junkie&quot; label is for addresses with 3 or more governance related labels (e.g. &quot;Optimism voter&quot;, &quot;RetroPGF Contributor and &quot;Delegator&quot;).

- &quot;Master Trader&quot; label is for addresses with all 3 elite trader labels i.e. Elite NFT Trader&quot;, &quot;Elite DEX Trader&quot; and &quot;Elite Perp Trader&quot; labels.

- &quot;Jack of All Trades&quot; label is for addresses that have all three trade related labels (i.e. NFT, Perp and DEX) but all are not elite trader labels.

- &quot;Prolific Bridge User&quot; label is for addresses with 3 or more bridge related labels. That is, addresses that have used 3 or more different bridges on Optimism.
