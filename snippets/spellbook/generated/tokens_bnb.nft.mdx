## tokens_bnb.nft 
Selection of NFT token addresses.


<ParamField path="name">
NFT Project Name
</ParamField>
<ParamField path="contract_address">
The contract address is the unique address of where the NFTs are stored.
</ParamField>
<ParamField path="standard">
The token standard. This is the set of rules that the contract must respect to be compatible with the common standards. Accepted values are presently standards represented. If you are adding a token with a new standard, remember to add it the accepted values list.

</ParamField>
