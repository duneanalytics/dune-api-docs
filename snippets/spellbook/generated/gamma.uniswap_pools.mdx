## gamma.uniswap_pools 
A canonical table containing Uniswap mappings for known Gamma pools created on Optimism. 


<ParamField path="blockchain">
Blockchain of the LP Token
</ParamField>
<ParamField path="project">
Project Name of the LP Token
</ParamField>
<ParamField path="lp_name">
Gamma LP Token name
</ParamField>
<ParamField path="contract_address">
Gamma LP Token contract address
</ParamField>
<ParamField path="pool_contract">
Uniswap liquidity pool contract address
</ParamField>
<ParamField path="token0">
Token0 contract address
</ParamField>
<ParamField path="token1">
Token1 contract address
</ParamField>
<ParamField path="fee">
Swap fee of a Uniswap liquidity pool. Divide by 1e4 or power(10,4) to get the fee percentage
</ParamField>
