## chainlink.price_feeds_hourly 
Hourly chainlink price feed updates


<ParamField path="blockchain">
Blockchain
</ParamField>
<ParamField path="hour">
Block time of the oracle update - truncated to hour
</ParamField>
<ParamField path="block_date">
Block Time, truncated to date
</ParamField>
<ParamField path="block_month">
Block time truncated to month
</ParamField>
<ParamField path="feed_name">
Feed name from Chainlink docs
</ParamField>
<ParamField path="oracle_price_avg">
Price returned by Chainlink oracle update
</ParamField>
<ParamField path="proxy_address">
Address of the proxy contract for the Chainlink oracle
</ParamField>
<ParamField path="aggregator_address">
Address of the aggregator contract tied to the proxy for the Chainlink oracle (via Read Contract)
</ParamField>
<ParamField path="base">
Base token of the pair
</ParamField>
<ParamField path="quote">
Quote token of the pair
</ParamField>
<ParamField path="underlying_token_price_avg">
Price of the underlying token, with extra decimal adjustments if needed
</ParamField>
