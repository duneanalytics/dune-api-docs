## aerodrome_base.bribe_mappings 
Bribe to Pool mappings for aerodrome on all chains


<ParamField path="blockchain">
Blockchain which the DEX is deployed
</ParamField>
<ParamField path="project">
Project name of the DEX
</ParamField>
<ParamField path="version">
Version of the contract built and deployed by the DEX project
</ParamField>
<ParamField path="pool_contract">
Address of the liquidity pool contract
</ParamField>
<ParamField path="pool_id">
Unique encoded identifier that refers to each pool
</ParamField>
<ParamField path="incentives_contract">
Address of the contract where incentives are stored and emitted.
</ParamField>
<ParamField path="incentives_type">
Description of the incentives address type.
</ParamField>
<ParamField path="evt_block_time">
Block time in UTC
</ParamField>
<ParamField path="evt_block_number">
Block Number
</ParamField>
<ParamField path="contract_address">
Gauge Creator Contract Address
</ParamField>
<ParamField path="evt_tx_hash">
Tx Hash
</ParamField>
<ParamField path="evt_index">
event index
</ParamField>
