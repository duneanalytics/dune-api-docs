## ens.resolver_records 
A table that shows the resolver addresses for known ENS names.


<ParamField path="name">
The ENS name
</ParamField>
<ParamField path="node">
The node, the namehash of the ENS name
</ParamField>
<ParamField path="address">
The resolver address of the ENS name
</ParamField>
<ParamField path="block_time">
block time of update
</ParamField>
<ParamField path="tx_hash">
transaction hash update
</ParamField>
<ParamField path="evt_index">
event index of node record update
</ParamField>
