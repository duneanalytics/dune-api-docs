## coinbase_wallet_quests_optimism.distributor_addresses 
A table containing quest information (i.e. name, distributor address) for Coinbase Wallet Quests


<ParamField path="distributor_address">
The address distributing quest rewards. There should be a distinct distributor address for each quest.
</ParamField>
<ParamField path="rewards_token">
The token used as a quest reward
</ParamField>
<ParamField path="quest_name">
The name of the quest
</ParamField>
