## prices.usd_daily 
Daily aggregated prices table across blockchains

<ParamField path="day">
UTC event block time truncated to the day mark
</ParamField>
<ParamField path="blockchain">
Native blockchain of the token
</ParamField>
<ParamField path="contract_address">
Contract address of the token
</ParamField>
<ParamField path="symbol">
Token symbol
</ParamField>
<ParamField path="decimals">
Token decimals
</ParamField>
<ParamField path="price">
daily average USD price of a token
</ParamField>
<ParamField path="price_open">
daily opening USD price of a token
</ParamField>
<ParamField path="price_high">
daily maximum USD price of a token
</ParamField>
<ParamField path="price_low">
daily minimum USD price of a token
</ParamField>
<ParamField path="price_close">
daily closing USD price of a token
</ParamField>
