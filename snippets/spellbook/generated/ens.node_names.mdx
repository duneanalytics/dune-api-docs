## ens.node_names 
A table that links a node (namehash) to a readable ENS name.


<ParamField path="node">
The node, the namehash of the ENS name
</ParamField>
<ParamField path="name">
The ENS name
</ParamField>
<ParamField path="label_name">
The name of the ENS label
</ParamField>
<ParamField path="label_hash">
The namehash of the ENS label
</ParamField>
<ParamField path="initial_address">
The initial resolver address when registering the ENS
</ParamField>
<ParamField path="tx_hash">
transaction hash
</ParamField>
<ParamField path="block_number">
block number
</ParamField>
<ParamField path="block_time">
block time
</ParamField>
<ParamField path="evt_index">
event index of the registration event
</ParamField>
