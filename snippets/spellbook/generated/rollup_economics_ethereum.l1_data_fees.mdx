## rollup_economics_ethereum.l1_data_fees 
Gas fees paid by rollups for posting transaction data or state updates on Ethereum

<ParamField path="name">
name of the rollup
</ParamField>
<ParamField path="hash">
hash of the l1 transaction
</ParamField>
<ParamField path="block_time">
UTC event block time of the transaction
</ParamField>
<ParamField path="block_number">
Block number of the transaction
</ParamField>
<ParamField path="data_length">
byte length of the data posted to l1
</ParamField>
<ParamField path="gas_spent">
ETH spent on posting data to L1
</ParamField>
<ParamField path="gas_spent_usd">
fee paid to post data on L1 in usd
</ParamField>
<ParamField path="gas_used">
Gas units used by L1 Transactions
</ParamField>
<ParamField path="calldata_gas_used">
Gas units used by calldata (input data) by L1 Transactions
</ParamField>
