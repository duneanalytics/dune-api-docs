## prices.usd_forward_fill 
View on prices that does a forward fill on the last 24h till the current timestamp

<ParamField path="minute">
UTC event block time truncated to the minute mark
</ParamField>
<ParamField path="blockchain">
Native blockchain of the token
</ParamField>
<ParamField path="contract_address">
Contract address of the token
</ParamField>
<ParamField path="symbol">
Token symbol
</ParamField>
<ParamField path="decimals">
Token decimals
</ParamField>
<ParamField path="price">
USD price of a token
</ParamField>
