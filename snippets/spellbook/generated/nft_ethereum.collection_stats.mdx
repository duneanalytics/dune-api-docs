## nft_ethereum.collection_stats 
This spell aggregates nft collections trading stats (volume, floor price) and pulls the data from nft.trades


<ParamField path="block_month">
Block month
</ParamField>
<ParamField path="block_date">
Block date
</ParamField>
<ParamField path="nft_contract_address">
NFT contract address
</ParamField>
<ParamField path="volume_eth">
daily nft collection volume Measured in ETH
</ParamField>
<ParamField path="trades">
daily nft collection number of trades
</ParamField>
<ParamField path="price_p5_eth">
5th percentile nft trade price Measured in ETH
</ParamField>
<ParamField path="price_min_eth">
Min NFT trade price Measured in ETH
</ParamField>
<ParamField path="price_max_eth">
Max NFT trade price Measured in ETH
</ParamField>
