## chain_info.chain_ids 
Table mapping chain IDs to names


<ParamField path="chain_name">
Full name of the Blockchain
</ParamField>
<ParamField path="chain_shortname">
Short name of the Blockchain
</ParamField>
<ParamField path="native_currency_symbol">
Symbol of the chain&#x27;s native currency
</ParamField>
<ParamField path="chain_id">
ID of the chain per EIP155
</ParamField>
<ParamField path="network_id">
ID of the network (usually the same as the Chain ID)
</ParamField>
<ParamField path="info_url">
URL with information about the blockchain
</ParamField>
<ParamField path="explorer">
URL for each chain&#x27;s main block explorer (first from the chainlist array)
</ParamField>
