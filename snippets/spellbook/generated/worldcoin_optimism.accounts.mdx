## worldcoin_optimism.accounts 
Known Worldcoin accounts (gnosis safes) on Optimism.


<ParamField path="worldcoin_deployer_address">
Worldcoin address that deployed the account (Safe)
</ParamField>
<ParamField path="account_address">
Worldcoin user account (Safe) address
</ParamField>
<ParamField path="created_time">
Created Time
</ParamField>
<ParamField path="creation_tx_hash">
Created Tx Hash
</ParamField>
<ParamField path="created_block_number">
Created Block Number
</ParamField>
<ParamField path="contract_project">
Project mapping for the contract. We expect this to be Gnosis Safe
</ParamField>
<ParamField path="contract_name">
Contract Name for the contract. We expect this to be a Gnosis Safe
</ParamField>
<ParamField path="trace_creator_address">
Deployer address in the creation trace. We expect this to be a Gnosis Safe deployer
</ParamField>
