## uniswap_v3_optimism.ovm1_pool_mapping 
A canonical table containing all Uniswap pool address changes between OVM1 and the EVM Equivalence regenesis release.


<ParamField path="oldAddress">
Uniswap liquidity pool contract address in OVM1
</ParamField>
<ParamField path="newAddress">
Uniswap liquidity pool contract address in the EVM Equivalence regenesis release.
</ParamField>
<ParamField path="token0">
Token0 contract address
</ParamField>
<ParamField path="token1">
Token1 contract address
</ParamField>
<ParamField path="fee">
Swap fee of a Uniswap liquidity pool. Divide by 1e6 or power(10,6) to get the fee percentage
</ParamField>
