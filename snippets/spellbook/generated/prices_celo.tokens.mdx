## prices_celo.tokens 
Price tokens on celo chain

<ParamField path="token_id">
Id of the token at coinpaprika. This id is required to pull the price feed data. NOTE: Not all tokens are listed at coinpaprika - consider using price data from DEX sources in this case or submit a listing request at coinpaprika.
</ParamField>
<ParamField path="blockchain">
Native blockchain of the token, if any
</ParamField>
<ParamField path="contract_address">
Contract address of the token, if any
</ParamField>
<ParamField path="symbol">
Token symbol
</ParamField>
<ParamField path="decimals">
Number of decimals for the token contract
</ParamField>
