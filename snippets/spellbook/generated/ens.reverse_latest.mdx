## ens.reverse_latest 
View the latest ENS reverse records


<ParamField path="address">
The ETH address
</ParamField>
<ParamField path="name">
The reverse ENS name
</ParamField>
<ParamField path="address_node">
The node (namehash) of the address reverse record
</ParamField>
<ParamField path="latest_tx_hash">
Latest transaction hash setting the reverse ENS
</ParamField>
<ParamField path="latest_tx_block_time">
Block time of the latest transaction hash setting the reverse ENS
</ParamField>
