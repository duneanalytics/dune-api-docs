## tokenlon.trades 
Tokenlon aggregator trades on all chains across all contracts and versions. This table will load dex_aggregator trades downstream.


<ParamField path="blockchain">
Blockchain which the DEX is deployed
</ParamField>
<ParamField path="project">
Project name of the DEX
</ParamField>
<ParamField path="version">
Version of the contract built and deployed by the DEX project
</ParamField>
<ParamField path="block_date">
UTC event block date of each DEX trade
</ParamField>
<ParamField path="block_month">
UTC event block month of each DEX trade
</ParamField>
<ParamField path="block_time">
UTC event block time of each DEX trade
</ParamField>
<ParamField path="token_bought_symbol">
Token symbol for token bought in the transaction
</ParamField>
<ParamField path="token_sold_symbol">
Token symbol for token sold in the transaction
</ParamField>
<ParamField path="token_pair">
Token symbol pair for each token involved in the transaction
</ParamField>
<ParamField path="token_bought_amount">
Value of the token bought at time of execution in the original currency
</ParamField>
<ParamField path="token_sold_amount">
Value of the token sold at time of execution in the original currency
</ParamField>
<ParamField path="token_bought_amount_raw">
Raw value of the token bought at time of execution in the original currency
</ParamField>
<ParamField path="token_sold_amount_raw">
Raw value of the token sold at time of execution in the original currency
</ParamField>
<ParamField path="amount_usd">
USD value of the trade at time of execution
</ParamField>
<ParamField path="token_bought_address">
Contract address of the token bought
</ParamField>
<ParamField path="token_sold_address">
Contract address of the token sold
</ParamField>
<ParamField path="taker">
Address of the taker
</ParamField>
<ParamField path="maker">
Address of the maker
</ParamField>
<ParamField path="project_contract_address">
Project contract address which executed the trade on the blockchain
</ParamField>
<ParamField path="tx_hash">
Unique transaction hash value tied to each transaction on the DEX
</ParamField>
<ParamField path="tx_from">
Address of the transaction sender
</ParamField>
<ParamField path="tx_to">
Address of the transaction receiver
</ParamField>
<ParamField path="trace_address">
Trace address of the transaction
</ParamField>
<ParamField path="evt_index">
Event index of the transaction
</ParamField>
