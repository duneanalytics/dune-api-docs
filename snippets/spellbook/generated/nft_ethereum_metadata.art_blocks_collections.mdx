## nft_ethereum_metadata.art_blocks_collections 
All art blocks collections and details (project id, project type, name, artist)


<ParamField path="contract_address">
Collection contract address
</ParamField>
<ParamField path="project_id">
Project ID. Can use to join with token_ids on contract address
</ParamField>
<ParamField path="project_id_base_value">
Value used to join on other tables
</ParamField>
<ParamField path="collection_name">
Name of collection
</ParamField>
<ParamField path="artist_name">
Name of artist
</ParamField>
<ParamField path="art_blocks_project_type">
Curated, Playground, Factory
</ParamField>
<ParamField path="art_blocks_curated_season">
If Curated collection, season it was launched in. Else N/A
</ParamField>
<ParamField path="art_collection_unique_id">
Unique identifier - contract address and project id
</ParamField>
