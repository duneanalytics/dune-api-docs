## addresses_events_testnets.first_funded_by 
Table showing who first funded each address in its native asset on every testnet chain

<ParamField path="blockchain">
Name of the blockchain
</ParamField>
<ParamField path="address">
Concerned address
</ParamField>
<ParamField path="first_funded_by">
Which address first funded the address with the native asset on that chain
</ParamField>
<ParamField path="block_time">
The block time on which the address was first funded with the native asset on that chain
</ParamField>
<ParamField path="block_number">
The block number on which the address was first funded with the native asset on that chain
</ParamField>
<ParamField path="tx_hash">
The transaction hash on which the address was first funded with the native asset on that chain
</ParamField>
