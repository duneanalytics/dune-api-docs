## nft_ethereum.aggregators_markers 
NFT aggregators markers


<ParamField path="hash_marker">
Hash used to identify an aggregator
</ParamField>
<ParamField path="aggregator_name">
Name of aggregator tied to the hash_marker
</ParamField>
<ParamField path="router_name">
Name of protocol that routed this trade using the aggregator, is NULL if it&#x27;s the same as aggregator_name
</ParamField>
<ParamField path="hash_marker_size">
Number of characters the hash_marker is made of, can be useful on joins
</ParamField>
