## tessera_ethereum.bids 
Bid entered into a Tessera Auction. There are different types of auction styles like LPDA.


<ParamField path="vault">
Address of the Vault
</ParamField>
<ParamField path="user">
Address of the end
</ParamField>
<ParamField path="type">
Type of Auction. For example, LPDA (Last Price Dutch Auction)
</ParamField>
<ParamField path="price">
Price per Rae of the Bid (ETH)
</ParamField>
<ParamField path="amount">
Amount of Raes in the bid
</ParamField>
<ParamField path="volume">
Value of the bid (ETH)
</ParamField>
<ParamField path="block_time">
Timestamp when the action took place
</ParamField>
<ParamField path="tx_hash">
Transaction hash for the transaction in which the action took place
</ParamField>
