## balances_polygon.matic_latest 
Latest token balances of matic for each address on polygon. Depends on transfers_polygon_matic.


<ParamField path="blockchain">
Blockchain
</ParamField>
<ParamField path="last_updated">
UTC timestamp when data was last updated
</ParamField>
<ParamField path="wallet_address">
Address of the wallet holding matic
</ParamField>
<ParamField path="token_address">
matic Dead Address on polygon
</ParamField>
<ParamField path="amount_raw">
Raw amount of matic token held *before* taking into account token decimals
</ParamField>
<ParamField path="amount">
Raw amount of matic token held *after* taking into account token decimals
</ParamField>
<ParamField path="amount_usd">
Amount of matic token held in USD if we have it
</ParamField>
<ParamField path="symbol">
Token symbol
</ParamField>
