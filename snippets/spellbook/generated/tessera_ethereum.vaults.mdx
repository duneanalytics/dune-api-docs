## tessera_ethereum.vaults 
Tessera Vault created. A Vault can contain multiple NFTs.


<ParamField path="deployer">
Address of the contract that deployed the Vault
</ParamField>
<ParamField path="origin">
Address of the originator
</ParamField>
<ParamField path="owner">
Address of the Vault owner at the time of deployment
</ParamField>
<ParamField path="vault">
Address of the Vault
</ParamField>
<ParamField path="block_time">
Timestamp when the action took place
</ParamField>
<ParamField path="tx_hash">
Transaction hash for the transaction in which the action took place
</ParamField>
