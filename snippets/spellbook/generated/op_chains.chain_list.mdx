## op_chains.chain_list 
Metadata about OP chains - Chains which build on the OP Stack and contirbute back to Optimism.

<ParamField path="blockchain">
Blockchain name as it appears in Dune schema. This should match exactly with Dune&#x27;s table schema (i.e. optimism)
</ParamField>
<ParamField path="blockchain_name">
User-Friendly Blockchain name, as it should appear in dashboards, visuals, etc (i.e. Optimism Mainnet)
</ParamField>
<ParamField path="chain_id">
Chain ID per EIP-155. See chainlist.org for a list.
</ParamField>
<ParamField path="launch_date">
Launch Date of the chain
</ParamField>
<ParamField path="is_superchain">
Binary flag if the chain is known to be Superchain aligned (does not exist yet - to be launched).
</ParamField>
