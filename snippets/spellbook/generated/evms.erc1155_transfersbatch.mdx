## evms.erc1155_transfersbatch 
Batch transfers events for ERC1155 tokens on EVM chains.

<ParamField path="blockchain">
Name of the blockchain
</ParamField>
<ParamField path="contract_address">
ERC1155 token contract address
</ParamField>
<ParamField path="evt_tx_hash">
Transaction hash of the event
</ParamField>
<ParamField path="evt_index">
Event index
</ParamField>
<ParamField path="evt_block_time">
Timestamp for block event time in UTC
</ParamField>
<ParamField path="evt_block_number">
Event block number
</ParamField>
<ParamField path="operator">
Addresses authorized (or approved) by a NFT owner to spend all of his token IDs.
</ParamField>
<ParamField path="from">
Wallet address that initiated the transaction
</ParamField>
<ParamField path="to">
Wallet address that received the transaction
</ParamField>
<ParamField path="ids">
ERC1155 token IDs
</ParamField>
<ParamField path="values">
Amounts of ERC1155 token transferred
</ParamField>
