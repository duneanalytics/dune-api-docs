## transfers_celo.erc20 
ERC20 Token Transfers on Celo. This table is updated every 30 minutes.

<ParamField path="blockchain">
Blockchain
</ParamField>
<ParamField path="transfer_type">
Transfer Type: sent/received
</ParamField>
<ParamField path="wallet_address">
Wallet address of sender or receiver. If amount is negative, wallet address is the sender&#x27;s.
</ParamField>
<ParamField path="token_address">
Contract address for token
</ParamField>
<ParamField path="block_time">
Timestamp for block event time in UTC
</ParamField>
<ParamField path="block_month">
Block Month column used to partition data in this table
</ParamField>
<ParamField path="amount_raw">
Raw amount of ERC20 token held *before* taking into account token decimals
</ParamField>
<ParamField path="evt_index">
Event Index
</ParamField>
<ParamField path="tx_hash">
Primary key of the transaction
</ParamField>
