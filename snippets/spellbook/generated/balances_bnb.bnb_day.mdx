## balances_bnb.bnb_day 
Daily token balances of bnb for each address on bnb. Depends on transfers_bnb_bnb.


<ParamField path="blockchain">
Blockchain
</ParamField>
<ParamField path="block_month">
UTC event block time truncated to the month mark
</ParamField>
<ParamField path="block_day">
UTC event block time truncated to the day mark
</ParamField>
<ParamField path="wallet_address">
Address of the wallet holding bnb
</ParamField>
<ParamField path="token_address">
bnb Dead Address on bnb
</ParamField>
<ParamField path="amount_raw">
Raw amount of bnb token held *before* taking into account token decimals
</ParamField>
<ParamField path="amount">
Raw amount of bnb token held *after* taking into account token decimals
</ParamField>
<ParamField path="amount_usd">
Amount of bnb token held in USD if we have it
</ParamField>
<ParamField path="symbol">
Token symbol
</ParamField>
