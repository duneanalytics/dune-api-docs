## tokens_optimism.nft_bridged_mapping 
Selection of NFT token addresses bridged from Ethereum to Optimism.


<ParamField path="contract_address">
The contract address is the unique address of where the NFTs are stored on Optimism.
</ParamField>
<ParamField path="name">
NFT Project name
</ParamField>
<ParamField path="standard">
NFT Project standard, currently only erc721 tokens can be bridged
</ParamField>
<ParamField path="symbol">
NFT Project symbol
</ParamField>
<ParamField path="contract_address_l1">
Bridged project contract address on L1
</ParamField>
