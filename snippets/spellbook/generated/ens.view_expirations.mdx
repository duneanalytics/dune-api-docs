## ens.view_expirations 
View expirations of ENS registrations


<ParamField path="label">
Hashed individual component of ENS name
</ParamField>
<ParamField path="min_expires">
Minimum expiration date UTC for domain
</ParamField>
<ParamField path="max_expires">
Maximum expiration date UTC for domain
</ParamField>
<ParamField path="min_evt_block_time">
Block time UTC of first name registration
</ParamField>
<ParamField path="max_evt_block_time">
Block time UTC of last name renewal
</ParamField>
<ParamField path="count">
Number of times expiration was extended
</ParamField>
