## yield_yak.deposits 
yield yak deposits


<ParamField path="blockchain">
Blockchain on which the YRT contract is deployed
</ParamField>
<ParamField path="contract_address">
Address of the YRT smart contract / strategy
</ParamField>
<ParamField path="tx_hash">
Transaction hash of the deposit/withdraw/reinvest
</ParamField>
<ParamField path="evt_index">
The position of this event log within this transaction
</ParamField>
<ParamField path="tx_index">
Index of the transaction within the block
</ParamField>
<ParamField path="block_time">
UTC time of the deposit/withdraw/reinvest
</ParamField>
<ParamField path="block_date">
UTC event block date of the deposit/withdraw/reinvest
</ParamField>
<ParamField path="block_number">
Block number of the deposit/withdraw/reinvest
</ParamField>
<ParamField path="user_address">
Address of the wallet which made the deposit/withdraw/reinvest
</ParamField>
<ParamField path="deposit_amount">
Amount of deposit tokens deposited into the strategy
</ParamField>
